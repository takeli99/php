Un PDO o también conocido como PHP data objects, es una extensión de PHP que ofrece una interfaz para acceder a bases de datos relacionales. Usa del mismo modo clases y métodos
que permiten interactuar con diversas bases de datos. Es más ligero y más rápido que SQL "normal".
Algunas de las ventajas:
    - Abstracciónde BBDD: Los desarrolladores pueden escribir código con diversas bases de datos sin cambiar la lógica de la aplicación.
    - Seguridad: Más seguridad para prevenir la inyección SQL.
    - Mayor compatibilidad: Gracias a la abstracción PDO facilita el cambio de BBDD sin necesidad de reescribir el código.
